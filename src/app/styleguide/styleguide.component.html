ApolloShip Styleguide

This styleguide is a collection of the components and patterns that make up ApolloShip.

With the possibility of many developers handling and editing the code, we wanted to create a living document to house the design patterns used. This can ensure not only a consistent UI but also consistent, readable code for anyone working on the app.
